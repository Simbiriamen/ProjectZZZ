## 🚀 Разработанная оптимизация: model_lightgbm_v2.py

### Ключевые улучшения:

#### 1. Расширенные признаки (+6 interaction features)
```python
- freq_trend_interaction = frequency_90d × group_trend_6m
- margin_popularity = margin × global_popularity
- recency_frequency = frequency_30d / (days_since_last_purchase + 1)
- share_trend = group_share_in_portfolio × group_trend_6m
- frequency_90d_sq (полиномиальный)
- days_since_log (логарифмический)
```

#### 2. Оптимизированные гиперпараметры
```yaml
num_leaves: 15 → 31      # +107% ёмкости
max_depth: 6 → 8         # +33% глубины
learning_rate: 0.1 → 0.05 # Лучшая сходимость
reg_alpha: 0 → 0.1       # L1 регуляризация
reg_lambda: 0 → 0.1      # L2 регуляризация
min_child_samples: 0 → 20 # Защита от переобучения
```

#### 3. Кросс-валидация (n_folds=3)
- Стабильный подбор числа итераций
- Защита от переобучения на валидации

#### 4. Ensemble калибровка
```
Platt Scaling (LogisticRegression)
    ↓
Isotonic Regression
    ↓
Взвешенное усреднение (сигмоида)
    ↓
Выбор лучшего по Brier Score
```

#### 5. Feature Selection
- Автоматическое удаление нижних 25% признаков по важности
- Уменьшение шума и ускорение инференса

#### 6. Улучшенный подбор порога
- Приоритет на Recall ≥ 0.55
- Поиск оптимального F1 в допустимом диапазоне
