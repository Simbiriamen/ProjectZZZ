ЭТАП 1: ПОДГОТОВКА ОКРУЖЕНИЯ И БАЗЫ ДАННЫХ
--------------------------------------------------------------------------------
1.1. Установка программного обеспечения
    - Установить Python 3.9 или выше (галочка "Add Python to PATH" обязательна).
    - Установить PostgreSQL (версия 14 или выше).
    - Установить pgAdmin 4 (идет в комплекте с PostgreSQL).
    - Установить Git (для работы с GitHub).

1.2. Создание папок проекта
    - Создать папку D:\ProjectZZZ.
    - Внутри создать подпапки: config, data, models, src, tests, docs.
    - В папке data создать: raw, processed, output.

1.3. Настройка Базы Данных (PostgreSQL)
    - Запустить pgAdmin 4.
    - Создать новую базу данных с именем: project_zzz_db.
    - Создать пользователя (если нет): имя postgres, пароль (запомнить!).
    - Проверить подключение через Python (библиотека psycopg2).

1.4. Создание таблиц в Базе Данных
    - Создать таблицы согласно архитектуре (см. файл schema.sql в приложении).
    - Основные таблицы: clients, skus, purchases, visits_schedule, visit_proposals.
    - Добавить таблицы для справочников (замены, аналоги, цены).
    - Настроить первичные ключи (Primary Key) и связи (Foreign Key).
